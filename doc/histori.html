Binär code

Jag ska prata om historian om binär kode kanse en av dom mest populär koder som många vet som code med bara ettor och nollor

Binär code var utväklar i 1600 tallet allafal det var den första gånge som vi vet säkärt att det blev utväklad men det var en tysk matemarticare namen G W Leibnitz som utväklade den men det var inte bara han men som tänkte att utväkla det men det var många matematicer som hade iden av en benär kode så G B Leibnitz var inte den ända men en va dom populars under tiden ent till var juan caramuel y lobkowitz han var en spansk och på något matemaatikenst sätt utväkadet dom binär koden mend eller utan leibniz men till slut kom det till datorena.
